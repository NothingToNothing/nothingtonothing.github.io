第二章 普通模式 笔记

31 Jul, 2018 · by Yamin Hu · Read in about 1 min · (76 Words)
vim 笔记

技巧 7: 停顿时请移开画笔

普通模式才是常态

技巧 8：把撤销单元切成块

  • 自由控制撤销的粒度
  • 插入模式到退出插入模式，是一次修改，可进行一次撤销u
  • 在插入模式中，如果使用了方向键的话，会进一步细分撤销粒度

技巧 9：构造可重复的修改

功能：光标位于单词最后一个字母，欲删除整个单词

方案：dbx，或bdw，或daw（不仅删除单词，还删除了单词前的空格）

最优方案：daw；原因，可以发挥.的最大威力。

  • db: 删除 光标位置到单词起始处的单词
  • b: 移动 光标到单词的起始处
  • aw: 文本对象a

技巧 10：用次数做简单的算术运算

  • [count]: 光标位置 或 其后 数字+count；不带count，为+1
  • yyp: 复制本行

技巧 11：能够重复，就别用次数

功能：删除两个单词

方案：d2w，2dw，dw.

最优方案：dw.，拥有更细的粒度，即所谓的，执行，重复，回退

技巧 12：双剑合璧，天下无敌

操作符 + 动作命令 = 操作

  • d{motion} : dl，字符；dw，单词；dap，段落
  • 操作符重复，作用于当前行
  • daw：删除单词
  • gg: 跳到文档开头
  • G: 跳到文档结
  • 操作符待决模式：dw，d键入后，进入待决模式

常见操作符

命令 用途
c 修改
d 删除
y 复制
g~ 反转大小写
gu 变小写
gU 变大写
> 增加缩进
< 减小缩进
= 自动缩进
! 使用外部程序过滤{motion}所跨越的行
