尝试Hugos

22 Jul, 2018 · by Yamin Hu · Read in about 1 min · (41 Words)
hubo blog

安装brew

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装Hugo

brew install hugo

创建站点

hugo new site MyBlog

增加theme

fork theme:hyde-y (从github到码云）

给站点增加theme

cd MyBLog
git init
git submodule add https://gitee.com/yaminhu/hyde-y.git themes/hyde-y

增加内容

根据hyde-y的一个live site的source code，修改content和data目录下的文件和config.toml。

生成静态网页

hugo --baseUrl="GITEE ADDRESS" -d DIR

上传静态网页到GitHub或码云

达到目的：不依赖GitHub或码云的解析功能
