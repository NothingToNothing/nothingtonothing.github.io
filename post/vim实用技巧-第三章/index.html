第三章 插入模式 笔记

1 Aug, 2018 · by Yamin Hu · Read in about 1 min · (58 Words)
vim 笔记

技巧 13：在插入模式中可即时更正错误

一个单词开头出错，删除整个单词，重新键入。

删除命令

命令 用途
<C-h> 删除前一个字符(同退格键)
<C-w> 删除前一个单词
<C-u> 删至行首

Note: 在命令行模式，和bash shell中，也可以使用，大赞。

技巧 14：返回普通模式

切换到普通模式：<C-[>

切换到插入-普通模式：<C-o>

插入-普通模式：执行完一条普通模式命令，立马回到插入模式

  • zz：当前行放在屏幕正中

技巧 15： 不离开插入模式，粘贴寄存器中的文本

  • set guifont=Menlo:h15：设置字体

  • set noimdisable：返回普通模式时，切换回英文输入法A

  • yt,:复制当前光标到,号前的所有字符

  • <C-r>{register}：粘贴

技巧 16：随时随地做运算

表达式寄存器

  • <C-r>=，输入表达式，

技巧 17： 用字符编码插入非常用字符

  • <C-v>065：输入A，A的ascii编码的十进制是65
  • <C-v>u00bf：输入¿，¿的unicode编码的十六进制是00bf
  • ga：光标所在位置字符的编码
  • <C-v>非数字键：按原义插入非数字字符

技巧 18：二合字母插入非常用字符

  • C-k?I：¿
  • C-k12：½
  • C-k<<：«
  • :h digraph-table：可用的二合字母表

技巧 19： 用替换模式替换已有文本

  • R: 普通模式进入替换模式
  • gR: 虚拟替换模式
  • r, gr: 替换一个字符后，马上回到普通模式
